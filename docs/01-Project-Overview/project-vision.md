# Project Vision

## Проблема

Разработчики, работающие над множеством проектов на разных языках (C#, Python, Rust, JavaScript и др.), сталкиваются с проблемой организации:

- Проекты разбросаны по всей файловой системе
- Нет единой точки входа для навигации
- Сложно отслеживать статус и дату последних изменений
- Полезные ссылки (видео, статьи, репозитории) теряются в закладках браузера

## Решение

**DevHub** — настольное приложение на C#/WPF, которое становится личным «центром управления разработкой».

### Ключевые возможности

| Возможность | Описание |
|-------------|----------|
| Список проектов | Удобный каталог всех проектов со статусами и тегами |
| Быстрый доступ | Открытие папки в Explorer, запуск IDE одной кнопкой |
| Захват ссылок | Глобальная горячая клавиша — ссылка автоматически попадает в приложение |
| Расширяемость | Модульная архитектура для будущих функций |

## Целевая аудитория

| Сегмент | Описание | Приоритет |
|---------|----------|-----------|
| Solo-разработчик | Один человек, 10+ проектов, разные языки | **Primary** |
| Фрилансер | Много клиентских проектов, нужна организация | Primary |
| Студент/энтузиаст | Учится, делает pet-projects, хочет порядок | Secondary |
| Малая команда | 2-5 человек, общий каталог проектов | Future |

## Ценности продукта

1. **Простота** — минимум кликов до действия
2. **Скорость** — мгновенный запуск, мгновенный поиск
3. **Локальность** — данные только на вашем ПК, никаких облаков
4. **Расширяемость** — растёт вместе с потребностями
