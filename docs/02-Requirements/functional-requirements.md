# Functional Requirements

> Приоритеты: **Must** — обязательно в MVP | **Should** — важно, но не критично | **Could** — nice-to-have

---

## FR-0xx — Управление проектами

| ID | Требование | Описание | Приоритет |
|----|------------|----------|-----------|
| FR-001 | Добавление проекта | Пользователь может добавить проект, указав: название, путь к папке, язык программирования, описание | **Must** |
| FR-002 | Редактирование проекта | Пользователь может изменить любое поле существующего проекта | **Must** |
| FR-003 | Удаление проекта | Пользователь может удалить проект с подтверждением | **Must** |
| FR-004 | Список проектов | Приложение отображает все проекты в виде списка/сетки с сортировкой | **Must** |
| FR-005 | Статус проекта | Каждый проект имеет статус: Active / Completed / Archived | **Must** |
| FR-006 | Теги проекта | К проекту можно привязать произвольные теги для фильтрации | **Should** |
| FR-007 | Заметки к проекту | Каждый проект содержит текстовое поле для заметок | **Should** |
| FR-008 | Избранное | Возможность пометить проект как избранный для быстрого доступа | **Could** |
| FR-009 | Импорт проектов | Автоматическое сканирование директории и импорт проектов | **Could** |

## FR-1xx — Навигация и доступ

| ID | Требование | Описание | Приоритет |
|----|------------|----------|-----------|
| FR-101 | Открыть в Explorer | Кнопка для открытия папки проекта в Проводнике Windows | **Must** |
| FR-102 | Открыть в IDE | Кнопка для запуска проекта в выбранной IDE (VS Code, Visual Studio, Rider) | **Must** |
| FR-103 | Настройка IDE | Пользователь может указать пути к IDE и выбрать IDE по умолчанию | **Must** |
| FR-104 | Копировать путь | Быстрое копирование пути к проекту в буфер обмена | **Should** |
| FR-105 | Открыть терминал | Кнопка для открытия терминала в папке проекта | **Could** |

## FR-2xx — Захват ссылок

| ID | Требование | Описание | Приоритет |
|----|------------|----------|-----------|
| FR-201 | Глобальная горячая клавиша | Перехват ссылки из буфера обмена по Ctrl+Shift+Y | **Should** |
| FR-202 | Хранение ссылок | Ссылки сохраняются в JSON с метаданными (URL, дата, теги) | **Should** |
| FR-203 | Категоризация ссылок | Ссылки можно привязать к проекту или к общей коллекции | **Should** |
| FR-204 | Предпросмотр ссылки | Автоматическое извлечение заголовка страницы по URL | **Could** |
| FR-205 | Поиск ссылок | Поиск сохранённых ссылок по URL, заголовку, тегам | **Should** |

## FR-3xx — Настройки и общее

| ID | Требование | Описание | Приоритет |
|----|------------|----------|-----------|
| FR-301 | Автозапуск | Приложение может запускаться вместе с Windows | **Should** |
| FR-302 | Сворачивание в трей | При закрытии окна приложение сворачивается в системный трей | **Should** |
| FR-303 | Тёмная/светлая тема | Переключение между темами оформления | **Could** |
| FR-304 | Экспорт данных | Экспорт списка проектов в JSON/CSV | **Could** |
| FR-305 | Резервное копирование | Автоматический бэкап JSON-файлов данных | **Should** |
