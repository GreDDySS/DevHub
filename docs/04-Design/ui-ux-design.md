# UI/UX Design

## Принципы дизайна

1. **Информационная плотность** — максимум полезной информации на экране
2. **Минимум кликов** — до любого действия не более 2 кликов
3. **Визуальная чёткость** — статусы, языки, теги визуально различимы
4. **Единообразие** — одинаковые паттерны во всех разделах

## Экраны

### 1. Главный экран — Список проектов

**Назначение:** Основной экран для навигации по проектам.

| Элемент | Описание |
|---------|----------|
| Sidebar | Навигация: Проекты / Ссылки / Настройки |
| Search bar | Поиск по названию проекта в реальном времени |
| Filter bar | Фильтр по статусу: All / Active / Completed / Archived |
| Project cards | Карточки с информацией и кнопками действий |
| Status bar | Общая статистика (количество проектов по статусам) |

**Карточка проекта:**

```
┌─────────────────────────────────────────────┐
│ {StatusColor}  {Name}    {Language}  {Status}  ⭐  │
│                                              │
│ {Path}                                       │
│ {Description (1 строка, обрезана)}           │
│                                              │
│ [📁 Explorer]  [💻 IDE]  [✏ Edit]  [🗑 Del] │
│                                              │
│ Теги: {tag1} {tag2}    Обновлено: {date}     │
└─────────────────────────────────────────────┘
```

## Иконки статусов

| Статус | Иконка | Цвет |
|--------|--------|------|
| Active | 🟢 | Зелёный |
| Completed | 🔵 | Синий |
| Archived | ⚪ | Серый |

## Клавиатурные сокращения

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+N` | Добавить проект |
| `Ctrl+F` | Фокус на поиск |
| `Delete` | Удалить выбранный проект |
| `Enter` | Открыть выбранный проект в Explorer |
| `Ctrl+Shift+Y` | Захват ссылки из буфера (глобальная) |
